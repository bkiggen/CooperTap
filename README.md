# The Tap (Mr. Cooper's Tap House)

## By Ben Kiggen

# Description

## A keg-maintenance system with both user and admin views

## Requirements

 * Node Package Manager

## Install

    $ git clone https://github.com/bkiggen/thetap
    $ cd thetap
    $ npm install

## Start Development Server

    $ npm start
